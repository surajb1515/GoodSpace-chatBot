# GoodSpace-chatBot

This is a chat bot which is made with openai API, socket.io, mongoDB and node JS

## How to set up the project
### Clone this project
```
git clone https://github.com/surajb1515/GoodSpace-chatBot.git 
```
```
npm i
```
### Change the .env file 
```
OPENAI_API_KEY=
MONGO_URL=
```
```
node app.js 
```
#### It will run on localhost:3000
